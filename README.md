# Role play page

## How to open?
### Solution 1
You open your command prompt and use the command `git clone https://github.com/DragosNH/RP_v2.git` inside the desired directory. Afterwards you open the `index.html` file inside a browser.

### Solution 2
You download the zip file from GitHub and you unzip it inside the desired folder and the you open `index.html` inside a browser.
